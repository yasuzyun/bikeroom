# bIkEroom

## サイト概要
### サイトテーマ
バイク用駐輪場の利用金額や場所がわかるコミュニティーサイト

### テーマを選んだ理由
自身がバイクで出かけた際に駐輪場を探すのに非常に困難したため

### ターゲットユーザ
バイクを利用する人

### 主な利用シーン
ツーリング、旅行
